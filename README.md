### Mario & Luigi’s pizzeria requirements
> API that given a type of pizza and a discount code we return:

* Ingredients.
* Price.
* Size.
* Percentage discount applied.

> The discount code for "opening" is 10 percent.
Discount codes are stored in uppercase

> When you can’t find the pizza or the account return http status code 404 (Not found)
